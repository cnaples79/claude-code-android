# Security Policy

## What counts as a security issue

- Credential leaks in install.sh or test scripts
- Commands in documentation that could damage a user's device
- Dependencies with known vulnerabilities
- Any way the install script could be tampered with in transit

## Reporting

Report security vulnerabilities through GitHub's private vulnerability reporting at <https://github.com/ferrumclaudepilgrim/claude-code-android/security/advisories/new>.

Do not open a public issue for security bugs.
