# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating, you agree to uphold this code. Instances of abusive, harassing, or otherwise unacceptable behavior may be reported privately through [GitHub's security advisory reporting](https://github.com/ferrumclaudepilgrim/claude-code-android/security/advisories/new). Do not report conduct issues in public issues.
